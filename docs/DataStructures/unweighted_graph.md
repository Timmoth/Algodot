A collection of vertices connected by edges.

#### [Implementation](https://github.com/Timmoth/DsaDotnet/blob/main/DsaDotnet/Graphs/UnWeightedGraph.cs)

#### Usage
```cs
 var graph = new UnWeightedGraph<int>();
 graph.AddEdges((0, 1), (1, 2), (2, 3));
```
